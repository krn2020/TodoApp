TodoApp — это веб-API для управления личными задачами, реализованное на .NET 8 с использованием Minimal API, трёхслойной архитектуры и PostgreSQL. 
Приложение поддерживает регистрацию, JWT-аутентификацию, создание задач, списки, подзадачи.

 Основные возможности

Регистрация и аутентификация с помощью **JWT-токенов**
Управление задачами (создание, редактирование, удаление, просмотр)
Списки задач для группировки
Подзадачи внутри каждой задачи
Пагинация и фильтрация

Проект построен по трёхслойной архитектуре:

API — слой представления (Minimal API Endpoints)
BLL — бизнес-логика (сервисы, DTO)
DAL — доступ к данным (Entity Framework Core, репозитории)

Используется паттерн Repository для абстракции работы с базой данных.
